# 🏥 Online Medicine Shopping System
A full-stack online medicine shopping system built with
Java Spring Boot and HTML/CSS/JavaScript.

---

## 🚀 Features

* ✅ User Registration and Login with JWT Authentication
* ✅ Browse and Search Medicines by Name
* ✅ Filter Medicines by Category
* ✅ Add Medicines to Shopping Cart
* ✅ Place Orders with Delivery Address
* ✅ Track Order Status
* ✅ Admin Dashboard for Management
* ✅ Responsive Frontend UI

---

## 🛠️ Technologies Used

* **Backend:** Java, Spring Boot, Spring Security
* **Database:** MySQL, Hibernate JPA
* **Frontend:** HTML, CSS, JavaScript
* **Authentication:** JWT Tokens
* **Build Tool:** Maven

---

## 🗄️ Database Tables

* users, categories, medicines, cart, orders, order_items

---

## ⚙️ How to Run

1. Install Java JDK 17 and MySQL 8.0+
2. Create database: `CREATE DATABASE online_medicine_shop;`
3. Update `application.properties` with your MySQL password
4. Run `.\mvnw.cmd spring-boot:run`
5. Open `http://localhost:8080`
